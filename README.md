# Nihar Ramoliya Portfolio

A personal responsive portfolio website to showcase skills, projects, and contact details.

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Features

- Modern responsive layout
- Skills and project showcase
- Contact section with email and phone links
- Smooth scrolling and scroll reveal animations

## Projects Highlighted

1. Lost & Found App
2. Portfolio Website
3. Student Performance Prediction
4. Compound Interest Calculator

## Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/niharr05/Portfolio.git
   ```
2. Open the project folder.
3. Run by opening `index.html` in your browser.

## Deploy on GitHub Pages

1. Push this project to a GitHub repository.
2. Open repository Settings.
3. Go to Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait for the live URL.
